# Day Six -  Creating a Table

## What I learned
A table is used to arrange and present data in rows and columns.
- caption : used to add caption to a table
- colgroup : groups a set of columns together
- thead : defines the header section of a table
- tfoot : defines the footer section of a table
- tbody : defines the body section of a table
- tr : represents a row within the table
- th : defines a table header cell
- td : defines a table data cell
- col : defines attributes for a table column
- border : draws border around the table
